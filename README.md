# mkpw Homebrew Tap

This is the Homebrew Tap for [mkpw](https://github.com/duffn/mkpw).

## Installation

- Option 1

```bash
brew install duffn/mkpw/mkpw
```

- Option 2

```bash
brew tap duffn/mkpw
brew install mkpw
```

- Option 3, in a [`brew bundle`](https://github.com/Homebrew/homebrew-bundle) `Brewfile`:

```ruby
tap "duffn/mkpw"
brew "mkpw"
```
